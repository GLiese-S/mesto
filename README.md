#Спринт №4 "Mesto: базовая функциональность" курса Веб-Разработчик Яндекс-Практикума.

Интерактивная страница, можно редактировать поля "Имя" и "О себе" в отдельной форме.

Используются адаптивная верстка и организация файлов по БЭМ, GRID, Векторная графика, Медиа-запросы, Относительные единицы измерения, шрифт ”Inter“, скрипт в формате JS. Дизайн размещен в Figma.
